Flutter 테스트를 실행하고 결과를 보고하라.

## Step 1: 테스트 실행
```bash
flutter test --reporter=expanded
```

## Step 2: 결과 분석
테스트 결과를 아래 형식으로 보고한다:
- 총 테스트 수, 통과/실패/스킵 수, 소요 시간
- 실패한 테스트가 있으면: 파일명, 테스트명, 에러 메시지, 수정 제안

## Step 3: 다음 단계 안내
- 모든 테스트 통과 → `/commit`으로 커밋 권장
- 테스트 실패 → 코드 수정 후 다시 `/test` 권장
